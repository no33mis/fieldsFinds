Fields and Finds

A Web Interface

Dear Explorer, welcome to the Fields and Finds Adventure.

Several archaeological objects have been found in agricultural fields near Edinburgh, Scotland. To ensure proper managing and maintaining of the valuable discoveries within different farmers land, a relational database has been set up. Additionally, a web interface providing the information from the database has been developed, in which our users can explore the different fields and finds. This interface provides both spatial (locational) and aspatial (other information) aspects.

And now it is your turn. Use the map, tables and several tabs and buttons to learn more about the environment. As changes may be made within the existing database, the maps and tables will automatically update to reflect the changes.

Technical requirements: A display resolution width of at least 1280px, lower display resolutions will result in overlay and distortion. Best experience in Google Chrome.
